Welcome to TrackStack!

This TrackStack website will be able to track your daily expenses.TrackStack is a web app for tracking daily expenses. Users can add, edit, and delete entries. Visualizations help users understand where and when they spend most. Big Picture: Help users take control of their spending habits. We are developing this project using VS Code (Python, HTML, CSS, ASP.NET, JavaScript).
